| Palabra clave | Posible obligación |

| historia, HU, requisito | Desarrollo, Documentación |

| soporte, ticket, GLPI | Soporte técnico |

| manual, guía, capacitación | Transferencia conocimiento |
| seguridad, vulnerabilidad | Seguridad |
| reunión, acta, transcripción | Reuniones |
